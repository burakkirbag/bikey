# BIKEY

---

## Setup

```
# Install backend dependencies
cd backend
npm install

# Install frontend dependencies
cd frontend
npm install

# Run backend
cd backend
npm run start

# Run frontend
cd frontend
npm run start

# Run tests
cd frontend
npm run test
```

## Used Technologies

- React and Typescript
- React Testing Library & Jest
- GraphQL
- JWT

## Features

- Bike list
- Filter by vehicle type
- search bike with id
- Tested with React Testing library
